Vite Build Setup Required

You are viewing the raw source code! Web browsers cannot run TypeScript (.tsx) natively. You must view the compiled site instead.

How to view the app successfully:
  • Option A — Running Locally:
    Do not click index.html. Open your Terminal inside this folder, run npm install then npm run dev, and open the URL displayed.
  • Option B — GitHub Pages Setup:
    Go to your GitHub repository -> Settings -> Pages.
    Under "Build and deployment" → "Source", select "GitHub Actions".
    This automatically runs our bundled workflow to compile and deploy your site perfectly!

(If you are using the AI Studio Live Preview iframe, the site is active and compiled automatically.)
